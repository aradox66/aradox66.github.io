    • Bernoulli/Binomial/Poisson/Geometric Distributions

    For random variable $X$:
    Expected value: $E(X) = \mu = \sum{x_{i}*P(X==x_{i})}$
    Variance: $\sigma^2 = \sum (x_{i}-\mu)^2 * P(X==x_{i})$

    Probability Mass Function: $f_{X} x$
    Cumulative Density Function: $F_{X} x$

    Bernoulli Distribution is binary (coin toss):
    $E(x) = p$
    $X \sim Bernoulli(p) $

    Binomial Distribution
    $P ($ $k$ successes after $n$ Bernoulli trials $)$
    for binomial variable $X \; Bin(n, p)$
    $P(X==k) = (^{n}_{k})p^{k}(1-p)^{n-k}$
    Reminder: $\left( _{r}^{n} \right) = \frac{n!}{(n-r)!r!}$
    scipy.stats.binom.pmf(k=k, n=n, p=p)
    Binomial CDF:
    $ P(X \leq k) = \displaystyle\sum_{i=0}^{k}(^{n}_{k})p^{k}(1-p)^{n-k} $
    scipy.stats.binom(n=n, p=p).cdf(k))
    Expected value: $E(X) = \bar{X} = \mu_{X} = np$
    Variance: $\sigma^2 = np(1-p)$

    Geometric Distribution (time until factory failure)
    $n$ Bernoulli trials until first success
    $X \sim \text{ or } X \sim G(p)$
    PMF: $ P(X==k) = p(1-p)^{k-1} \text{ for } 1\leq k \leq \infty $
    CDF: $ P(X \leq k) = 1 - (1-p)^{k} \text{ for } 1\leq k\leq \infty $
    $\mu=\frac{1}{p}$
    $\sigma^{2}=\frac{1-p}{p^2}$

    Poisson Distribution (how many shooting stars)
    Probability of $k$ (# independent events $\left|\{E_{i}\}\right|$) within a fixed period given rate of occurence $\lambda$.
    $ X \sim \text{Po}(\lambda)$
    PMF: $ P(X==k) = \frac{\lambda^{k} e^{-\lambda}}{k!} $
    scipy.stats.poisson.pmf(k=k, mu=lambda)
    CDF: $ P(X\leq k) = \displaystyle\sum^{k}_{i=0}\frac{\lambda^{k} e^{-\lambda}}{k!}$
    $\bar{X}=\mu_{X}=\lambda$
    $\sigma^{2}_{X}=\lambda$
